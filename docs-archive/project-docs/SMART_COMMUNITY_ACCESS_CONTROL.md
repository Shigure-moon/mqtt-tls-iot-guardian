# 智慧社区门禁系统 (Smart Community Access Control)

## 项目概述

基于现有 MQTT TLS IoT Guardian 平台，开发智慧社区门禁系统，实现安全可靠的设备通信和OTA固件更新。

## 技术栈

- **硬件**: ESP32 + 读卡器/人脸识别模块
- **通信**: MQTT over TLS (使用现有证书体系)
- **固件更新**: OTA with 固件签名验证
- **后端**: FastAPI + PostgreSQL + Redis
- **前端**: Vue.js + Element Plus
